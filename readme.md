# Webhmi

Web based touch HMI to control different robots

![HMI](static/images/TouchHmi.png)

## Requirements
- Python3: `sudo apt install python3`
- Pip: `sudo apt install python3-pip`
- Flask: `pip install Flask `
- OpenCV: `pip install opencv-python`

## Usage
    python3 app.py
